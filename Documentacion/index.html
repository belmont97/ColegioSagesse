Descripción del Sistema.

El sistema implementado en este proyecto es de tipo administrativo, ya que, será manipulado solamente por el director (a) del colegio. Este sistema está desarrollado con el lenguaje de programación de HTML, trabajado en el editor de Visual Studio Code, como parte de la EE de Sistemas Web.

El sistema se conecta a una base de datos que será manipulada a través del sistema, dado que, es donde se realizan las altas, bajas y modificaciones de la base de datos.

El administrador puede acceder al sistema a través de un logueo, una vez verificado con los datos correspondientes, el sistema le permite agregar un maestro nuevo, consultar los maestros existentes en la base de datos, modificar los datos de los maestros o eliminar de la base de datos al maestro.

El sistema también esta disponible para los usuarios generales que deseen conocer un poco sobre el Colegio Sagesse, estos usuarios solamente podrán ver la información.

Manual de Instalación

1.0 Introducción

El objetivo de este manual es servir de guía de instalación y ejecución del sistema de administración del Colegio Sagesse, para el maestro de la EE Sistemas Web. Puede acceder también desde la web, ya que, el sistema fue desplegado con Heroku. El sistema permite como ya se mencionó la gestión de los maestros de la institución a través de una base de datos y, además, la visualización de la información por parte de los usuarios.

El manual se compone de las siguientes partes Instalación del sistema Instalación de la base de datos Ejecución del sistema

2.0 Instalación del Sistema

En este apartado se explicará como debe de instalar el sistema para que pueda tenerlo en su computadora y desplegarlo.

2.0 Instalación del Sistema

2.1 Requerimientos del sistema

Para el correcto funcionamiento del sistema se requiere tener una cuenta de GitHub, ya que, en este sitio se encuentra el repositorio que contiene el proyecto y deberá ser clonado, también se debe tener el programa de Visual Studio Code instalado en su computadora, para tener ahí el proyecto clonado.

Entonces para poder disponer de estos programas la computadora deber tener requerimientos mínimos:

Procesador de 1,6 GHz o más rápido 1 GB de RAM

También es necesario disponer del Sistema Gestor de Base de Datos MySQL, para la creación de la base de datos local. Siendo así también se requieren de algunos requerimientos mínimos para instalar este Sistema Gestor:

512 Mb de memoria RAM. 1024 Mb máquina virtual. 1 GB de espacio de disco duro. Sistema operativo: Windows, Linux y Unix. Arquitectura del sistema 32/64 bit. Protocolo de red TCP/IP.
